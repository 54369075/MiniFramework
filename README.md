### 2018.7.27日志
* 目前框架核心模块支持对象序列化（二进制序列化、Xml序列化、Json序列化、ProtoBuff序列化），目前Vector类型的序列化只有XML方式支持
* 消息分发
* 对象池（支持Prefab）
* 单例模板（支持继承MonoBehaviour的类）
* 压缩和解压
* 异步逻辑（支持链式调用）
* 有限状态机。
